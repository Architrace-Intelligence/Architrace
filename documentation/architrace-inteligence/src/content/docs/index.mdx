## What Architrace does

Architrace converts runtime telemetry into architecture intelligence.

1. Receives traces from OpenTelemetry collectors on OTLP gRPC.
2. Converts spans into graph nodes and edges.
3. Streams graph batches to a control-plane over bidirectional gRPC.
4. Applies control-plane configuration updates to active agents.
